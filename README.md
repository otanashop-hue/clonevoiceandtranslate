پروژه کلون کردن صدا و ترجه از فارسی به انگلیسی و خواندن متن انگلیسی از فایل SRT و صحبت کردن و ایجاد یک ویس براساس فایل طوریکه صحبت ها براساس زمانبندی که در فایل هست باشه . یعنی یک فایل صوتی به زبان فارسی بدم و یک فایل انگلیسی با صدای خودم خروجی بگیرم طوریکه از نظر زمانی مثل هم باشن

## قابلیت‌ها
- تولید SRT دو زبانه: برای هر زیرنویس، خط اول انگلیسی، خط دوم فارسی
- کلون کردن صدا با Coqui XTTS v2 و تولید صدای انگلیسی هم‌زمان با زمان‌بندی SRT
- پیاده‌سازی ساده با Faster-Whisper برای تبدیل گفتار فارسی به متن

## نصب
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```

برای CUDA، پکیج `torch` مناسب سیستم خود را نصب کنید.

## اجرا
```bash
python -m app.cli /path/to/input_fa_audio.wav --speaker-ref /path/to/your_voice_ref.wav --out-dir outputs --device cpu
```
خروجی‌ها:
- فایل `*.bilingual.srt` شامل متن دو زبانه (انگلیسی + فارسی)
- فایل `*.en.cloned.wav` شامل صدای انگلیسی هم‌زمان با زیرنویس‌ها
