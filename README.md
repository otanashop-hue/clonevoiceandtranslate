# Multi-Symbol Multi-Timeframe Signal Generator

این پروژه یک اندیکاتور Pine Script است که تابع `f_Vanguard()` شما را روی 20 نماد مختلف و 20 تایم فریم مختلف اجرا می‌کند و سیگنال‌های خرید و فروش را تولید می‌کند.

## ویژگی‌ها

- **20 نماد قابل تنظیم**: BTCUSDT, ETHUSDT, ADAUSDT و غیره
- **20 تایم فریم مختلف**: از 1 دقیقه تا 1 ماه
- **سیگنال‌های مختلف**: BUY, SELL, BUY60, SELL60
- **سیستم آلرت**: هشدار خودکار برای سیگنال‌ها
- **جدول نمایش**: نمایش سیگنال‌های اخیر در جدول
- **نمایش بصری**: شکل‌های هندسی روی چارت

## نحوه استفاده

### 1. نصب در TradingView

1. فایل `multi_symbol_multi_timeframe_signals_final.pine` را کپی کنید
2. در TradingView، روی "Pine Editor" کلیک کنید
3. کد را paste کنید
4. روی "Add to Chart" کلیک کنید

### 2. تنظیمات

#### نمادها (Symbols)
- 20 فیلد برای وارد کردن نمادهای مورد نظر
- به صورت پیش‌فرض: BTCUSDT, ETHUSDT, ADAUSDT و غیره

#### تایم فریم‌ها (Timeframes)
- 20 فیلد برای تنظیم تایم فریم‌ها
- فرمت‌های پشتیبانی شده:
  - دقیقه: "1", "3", "5", "15", "30", "60", "240", "480"
  - روزانه: "1D", "3D"
  - هفتگی: "1W"
  - ماهانه: "1M"

#### تنظیمات آلرت
- **Enable Alerts**: فعال/غیرفعال کردن آلرت‌ها
- **Show Signal Table**: نمایش جدول سیگنال‌ها
- **Max Signals to Display**: حداکثر تعداد سیگنال‌های نمایش داده شده

### 3. انواع سیگنال‌ها

- **BUY (1)**: سیگنال خرید 30 دقیقه‌ای
- **SELL (2)**: سیگنال فروش 30 دقیقه‌ای  
- **BUY60 (3)**: سیگنال خرید 60 دقیقه‌ای
- **SELL60 (4)**: سیگنال فروش 60 دقیقه‌ای

### 4. نمایش بصری

- **مثلث سبز کوچک**: سیگنال BUY
- **مثلث قرمز کوچک**: سیگنال SELL
- **مثلث سبز بزرگ**: سیگنال BUY60
- **مثلث نارنجی بزرگ**: سیگنال SELL60

## ساختار کد

### تابع اصلی f_Vanguard()

این تابع شامل منطق کامل تحلیل تکنیکال شما است:

```pinescript
f_Vanguard() =>
    // محاسبه EMA ها
    ema1 = ta.ema(hl2, 30)
    ema2 = ta.ema(hl2, 60)
    ema3 = ta.ema(hl2, 144)
    ema4 = ta.ema(hl2, 240)
    
    // محاسبه RSI و CCI
    rsi = ta.rsi(close, 14)
    cci = ta.cci(hlc3, 20)
    
    // منطق سیگنال‌های خرید و فروش
    // ...
    
    // خروجی: 1=BUY, 2=SELL, 3=BUY60, 4=SELL60, 5=بدون سیگنال
    sig
```

### سیستم Multi-Symbol Multi-Timeframe

```pinescript
// بررسی سیگنال برای هر نماد/تایم فریم
check_signal(symbol, timeframe) =>
    [sig] = request.security(symbol, timeframe, [f_Vanguard()])
    // پردازش سیگنال و ارسال آلرت
```

## محدودیت‌ها

1. **محدودیت درخواست‌ها**: TradingView محدودیت 40 درخواست `request.security()` در هر بار دارد
2. **عملکرد**: با افزایش تعداد نمادها و تایم فریم‌ها، عملکرد کاهش می‌یابد
3. **تاخیر**: سیگنال‌ها ممکن است با تاخیر نمایش داده شوند

## بهینه‌سازی

برای بهبود عملکرد:

1. تعداد نمادها و تایم فریم‌ها را کاهش دهید
2. فقط تایم فریم‌های مهم را انتخاب کنید
3. از `max_signals_display` برای محدود کردن نمایش استفاده کنید

## مثال استفاده

```pinescript
// تنظیم نمادها
symbol1 = "BTCUSDT"
symbol2 = "ETHUSDT"
// ...

// تنظیم تایم فریم‌ها  
timeframe1 = "1"    // 1 دقیقه
timeframe2 = "5"    // 5 دقیقه
timeframe3 = "15"   // 15 دقیقه
// ...

// فعال کردن آلرت‌ها
enable_alerts = true
```

## عیب‌یابی

### مشکل: سیگنال‌ها نمایش داده نمی‌شوند
- بررسی کنید که نمادها درست وارد شده باشند
- مطمئن شوید که تایم فریم‌ها معتبر هستند
- آلرت‌ها را فعال کنید

### مشکل: عملکرد کند
- تعداد نمادها و تایم فریم‌ها را کاهش دهید
- از تایم فریم‌های کوتاه‌تر استفاده کنید

### مشکل: آلرت‌ها کار نمی‌کنند
- در TradingView، آلرت‌ها را به صورت دستی تنظیم کنید
- مطمئن شوید که `enable_alerts = true` است

## پشتیبانی

برای سوالات و مشکلات:
1. بررسی کنید که کد به درستی کپی شده باشد
2. تنظیمات را دوباره بررسی کنید
3. از نسخه‌های مختلف فایل استفاده کنید
